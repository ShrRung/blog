杭州麦德魔方企业管理有限公司 （MAIDE MAGIC） 成立于2015年8月，是一家将音乐培训智能化的科技公司，公司三大核心产品为：乐器培训IVT系统(虚拟智能教学：Intelligent virtual teaching）、麦德魔方O2O平台、麦德魔方线下旗舰店。“智能化、趣味化、专业化学习音乐”是麦德魔方的教学理念。麦德魔方公司首创利用裸眼3D、VR等高科技技术结合O2O教育平台将总部的优质教育资源进行整合，形成人工智能设备研发、生产、软件开发、在线视频课堂、线下麦德魔方旗舰店的平台运营产业链。2015年11月第一款智能音乐教学系统‘麦德魔方1.0’问世，通过全国范围内进行市场调研和推广，得到了众多音乐体验者、音乐教师、家长及儿童的好评，论证了麦德魔方的研发理念。 麦德魔方品牌理念：智能音乐教学专家。
麦德魔方线下旗舰店定位：做乐器综合培训的劳斯莱斯。
麦德魔方旗下品牌：FUNK消音鼓、FUNK鼓校、IPA国际打击乐协会、GC、时光机器。
镲片系列：Soultone、MEMORY、CARSH、Curkey、动感领域、自由之神、灵动等系列。
麦德魔方虚拟智能音乐教学系统创始人：翟麦德（曾用名：翟高科，原杭州方克教育科技有限公司创始人）
